谈谈自己

  • 我是一名广州的小小大专生，努力的学习着网站知识还有Android方面的知识，每天学了这么多的知识，虽然在课堂上有很多很多的实操机会，但是都知识一些基础的知识，想进一步的深入要靠自己去找资料学习。奈何找到的不一定是自己想要的。所以一直想通过博客来记录下自己的学习历程，不仅提升自己的总结能力，加深自己的记忆，还可以分享给别人。记得国外的必修课就是但是逛过各种各样的博客提供网站之后觉得都不是想要的（虽然那些网站可以提供客观的流量支持，就是通过那些能让更多的用户去看到你的博客）。
  • 直到我我某一天看到了Hexo之后，被那漂亮的网站外观所吸引（一不小心暴露了自己是作品外貌协会的，当然啦，对作品的外观要求高也是人之常情，所以见异思迁也是无可厚非之后，我决定我也弄一个来写写自己的学习历程也不错
  • 还有一个原因是因为之前看过一篇CSDN的日报说明了写作的重要性。（详细点击）
  • 最后，要感谢我现在使用的主题Indigo的作者王昱森，也感谢给我解答问题的温斯渤

    跟我来了解Hexo

    官网介绍：

  • 快速、简洁且高效的博客框架
  • Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。
  • Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。
  • 只需一条指令即可部署到 GitHub Pages, Heroku 或其他网站。
  • Hexo 拥有强大的插件系统，安装插件可以让 Hexo 支持 Jade, CoffeeScript。

如果觉得上面的简洁介绍觉得不够的话可以去Hexo的官网查看API.或者有什么问题可以通过主页的邮箱发邮件或者通过GitHub发issue给我。我会尽我所能帮你解决。我觉得对于大部分的程序员来说，只要是自己喜欢的作品，自己会不惜一切代价去做出来。

关于Indigo

Indigo使用了类似Android的MD设计模式，几乎在Google自家的网页设计中，这种设计语言运用自如，但是运用在博客网站中我却是第一次见到的，所以才会有想要试试的心态。
