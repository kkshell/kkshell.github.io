安装Hexo所需要的所有软件

安装Git

  • 下载并安装Git 软件安装后最好配置一下环境变量，否则有些命令是不能在Git Bash中使用的
  • 本教程使用的是GitHub+Hexo来建立个人网站。所以您还没有GitHub账号的话请注册
  • 创建好个人账号之后，就可以自由的clone或者下载其他项目，但是你自己的网站是无法提交的。因为想要提交到GitHub上一定要通过授权，而GitHub是基于SSH（什么是SSH？）授权的。
  • 大家可以在安装好Git之后再桌面点击右键Git Bash Here来运行Git Bash终端来安装SSH。

  • 在安装SSH之前先用以下命令设置用户名和邮箱.

$ npm install hexo-renderer-less –save

$ npm install hexo-generator-feed –save

$ npm install hexo-generator-json-content –save

$ npm install hexo-helper-qrcode –save

layout: tags
comments: false

layout: categories
comments: false
```

配置404页面

将404页面放在blog/public下，就放这里就行了，网上很多人说放在source下，亲试不可用！

检验实践成功

打开浏览器访问http://localhost:4000/就可以啦！o(*￣▽￣*)o 如果一直打不开网站，可以用命令hexo server -p 5000更改端口号，新打开网址http://localhost:5000/
